Uso do Programa:
	No terminal, no mesmo diretório que o arquivo Makefile, é possível:
	- digitar o comando "make" para compilar o programa;
	- digitar o comando "make run" para executar o programa;
	- digitar o comando "make clean" para limpar os códigos objeto.
