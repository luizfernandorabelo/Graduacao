# Palíndromos

Uma frase é um palíndromo se a sequência de letras que a forma é a mesma, quer seja lida da esquerda para a direita ou da direita para a esquerda. Neste exercício, implemente um algoritmo **com uso de pilhas** para reconhecer se uma frase é um palíndromo. Sinais de pontuação devem ser desconsiderados para o reconhecimento de um palíndromo. Letras em maiúscula devem ser considerados em maiúscula.
